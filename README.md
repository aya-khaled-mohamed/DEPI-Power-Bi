An enterprise-grade, end-to-end clinical and financial intelligence project utilizing synthetic patient records generated by **Synthea**. This collaborative project covers the entire data analytics lifecycle: secure programmatic extraction from **SQL Server**, comprehensive data cleaning, complex relational data modeling, advanced DAX/measures engineering, and professional visual delivery across **Power BI**, **Tableau**, and **Microsoft Excel**.

---

## 📌 Project Overview

Modern healthcare organizations require data-driven insights to monitor patient demographics, optimize clinical encounters, evaluate treatment plans, track procedure expenses, and secure immunization coverage. 

By analyzing the comprehensive Synthea healthcare dataset, our team built unified, interactive executive dashboards that translate complex relational clinical tables into actionable strategic insights.

### 🔑 Key Analytical Pillars
* **Patient Demographics & Access:** Geolocation patterns, age distribution, and socio-demographic clinical profiling.
* **Clinical Encounters & Conditions:** Admission frequencies, top diagnoses, and patient encounter lengths.
* **Therapeutics & Immunizations:** Medication compliance, active prescriptions, and vaccine distribution trends.
* **Procedures & Financial Analysis:** High-cost clinical procedures, surgery trends, and cost efficiency.
* **Patient Safety & Careplans:** Active patient care plans, chronic disease management, and allergy profiling.

## 🚀 Technical Pipeline & Implementation

1. **Database Connection & SQL Extraction:**
   * Programmatic, secure database connections were implemented to securely extract and load raw, multi-table relational schemas directly from a centralized SQL Server.
2. **Data Cleansing & Transformation:**
   * Executed robust SQL/Power Query scripts to standardize date-time dimensions, clean null categorical inputs, handle missing data points, and segment metrics (e.g., age-binning, blood pressure thresholds).
3. **Data Modeling:**
   * Designed and configured a high-performance relational database schema (Star/Snowflake hybrid) mapping fact and dimension tables securely to avoid logical circular dependencies.
4. **Measures & Calculations Engineering (DAX & SQL):**
   * Formulated advanced analytical KPIs, cost calculations, time-intelligence trends, and year-over-year metrics to power downstream reporting.
5. **Dashboard UI/UX and Responsive Design:**
   * Developed clean, interactive executive interfaces utilizing consistent typography, professional dark/light thematic layouts, and dedicated responsive **Mobile Layouts** for on-the-go viewing.
